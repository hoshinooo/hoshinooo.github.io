Window系统安全

磁盘分区管理

磁盘的分区管理

Win+R运行，执行 diskmgmt.msc 打开磁盘管理

  • 右击分区 => 格式化

  • 格式化将清除卷上的所有数据

更改驱动器号

更改驱动器盘符

  • 使用驱动器号来表示并访问分区或卷

发布共享文件夹

Windows共享概述

微软公司推出的网络文件/打印机服务系统

  • 可以将一台主机的资源分布给其他主机共用
  • 共享=资源复用

共享访问的优点

  • 方便、快捷
  • 相比光盘/U盘不易受文件大小限制
  • 可以实现访问控制权限

配置共享文件夹

右击文件夹 => 共享 => 添加用户 => 分配权限

访问共享文件夹

通过UNC路径访问

UNC（Universal Naming Convention）通用命名规则，快捷访问网络中资源

以下位置可输入UNC路径

  • 资源管理器的地址栏
  • Win+r，运行框

UNC路径格式

  • \\服务器名\[共享名]
  • \\服务器IP\[共享名]

通过映射网络驱动器访问

把网络共享资源映射为网络驱动器（盘符），以简化访问步骤，常用于多频次访问场景

映射网络驱动器

  • 右击 计算机 或 网络
  • cmd运行，net use X: \\服务器地址\共享名

断开网络驱动器

  • 右击网络驱动器，选择 断开

共享资源管控

计算机管理-共享文件夹

管理共享文件夹

  • 开始 => 管理工具 => 计算机管理 => 系统工具 => 共享文件夹 => 共享

监控及管理已连接的用户

  • 开始 => 管理工具 => 计算机管理 => 系统工具 => 共享文件夹 => 会话

监控及管理被打开的文件

  • 开始 => 管理工具 => 计算机管理 => 系统工具 => 共享文件夹 => 打开的文件

特殊共享

特殊共享

  • 系统自动建立的用于管理目的的共享文件夹

主要类别

  • Drive letter$
  • ADMIN$

隐藏共享

  • 共享名$

FTP服务概述

概述

FTP（File Transfer Protocol，文件传输协议）

  • 是一种用于在网络上进行文件传输的协议
  • 通俗说就是实现文件共享功能

基于C/S架构的文档下载/上传通信

  • FTP基于客户端-服务器模型，其中一个计算机作为FTP服务器，而其他计算机作为FTP客户端
  • 服务端程序：提供文件存储和传输服务
  • 客户端程序：通过连接到FTP服务器进行文件上传和下载

FTP服务组件

Microsoft IIS

  • Internet Information Service，互联网信息服务
  • 微软提供的Web/FTP服务产品，集成在Windows操作系统中

部署FTP服务

安装FTP服务软件

安装核心组件

  • 右击 此电脑 => 管理 => 管理 => 添加角色和功能 => 默认下一步 => … => Web服务器（IIS） => 添加功能 => 下一步… => FTP服务 => 安装

添加FTP站点

添加FTP站点，指定站点名及根目录

FTP身份验证

FTP支持匿名登录和基本（认证）登录两种身份验证方式

  • 匿名登录允许用户以匿名身份登录FTP服务器，并访问公共文件
  • 基本认证登录要求用户提供用户名和密码进行身份验证，只有通过验证的用户才能够访问受限文件
  • 匿名身份验证，客户端通过资源管理器访问FTP直接登录
  • 基本身份验证，客户端通过资源管理器访问FTP需输入用户名及密码

常用FTP客户端软件

Windows资源管理器

  • 资源管理器地址栏输入ftp://服务器IP

专用图形工具

  • FileZilla Client、FlashFXP、LeapFTP、CuteFTP

网页浏览器（仅下载）

  • 打开浏览器地址栏输入ftp://服务器IP
